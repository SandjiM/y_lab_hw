# Product Catalog Service (Console, Java Core + Collections)

Консольное приложение для управления каталогом маркетплейса.  
Технологии: **Java Core** и **коллекции**, без сторонних библиотек.  
Поддержка: авторизация (ADMIN/VIEWER), CRUD товаров, поиск/фильтры, кэш запросов, аудит действий, метрики, **персистентность в TSV**.

## Быстрый старт

### Требования
- JDK 17+ (рекомендовано 17 или 21)
- IntelliJ IDEA / любой терминал с `javac` и `java`


### Запуск в IntelliJ
1. Открыть проект `product-catalog-service`.
2. Запустить `Run → Run 'Main'` для класса `app.cli.Main`.
3. При первом старте будут созданы файлы в `data/`:
    - `users.tsv` — пользователи по умолчанию,
    - `products.tsv` — появится после первого CRUD,
    - `audit.log` — заполняется по мере действий.

### Запуск из терминала
```bash
# из корня проекта
javac -d out $(find src -name "*.java")
java -cp out app.cli.Main
```

### Учётные записи по умолчанию

При первом запуске создаётся data/users.tsv с двумя пользователями:

Админ

- Логин: admin

- Пароль: admin

 - Роль: ADMIN

Зритель

 - Логин: viewer

 - Пароль: viewer

 - Роль: VIEWER

Если users.tsv повреждён, удалите файл и перезапустите — он пересоздастся с данными выше.
